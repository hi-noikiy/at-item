交易所 : 币种配置

Created by xingmingfeng, last modified on Mar 17, 2018

币种配置：CoinSymbol.java

原方法方法说明新方法
CoinSymbol fromValue(String value)通过 value 查找对应 币种，例：value="btc" 
CoinSymbol fromName(String name)通过 name 查找对应 币种，例：value="BTC" 
List<String> getStartCoinString()获取所有 已启用 的币种的 value 值（大写） 
List<CoinSymbol> getStartCoinSymbol()获取所有 已启用 的 币种 
List<CoinSymbol> getCountCoinSymbol()获取所有 已启用 的 计价货币 
List<CoinSymbol> getCryptoCoinSymbol()获取所有 已启用 的 数字货币 
List<CoinSymbol> getCurrencyCoinSymbol()获取所有 已启用 的 法币 
String getCurrencyCoinStr()获取所有 已启用 的 法币 的 value 值的 字符串 形式，用 逗号 分隔 
String getCryptoCoinStr()获取所有 已启用 的 数字货币 的 value 值的 字符串 形式，用 逗号 分隔 
List<String> getCurrencyCoinSymbolString()获取所有的 法币 
List<String> getCryptoCoinSymbolString()获取所有 已启用 的 数字货币 的 name 值 
List<CoinSymbol> getIsAllRateSymbol()获取所有 已启用 的 需要计算与其他所有币种的汇率 的 币种 
List<CoinSymbol> getSymbolExpectValue(String value)获取所有 已启用 的 除 value本身对应的币种 外的 币种
