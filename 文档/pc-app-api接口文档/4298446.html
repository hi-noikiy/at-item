List<Integer> getComFeeTypes(String coinSymbol)根据 币种 获取对应的币种 全部 的 公司手续费二级账户 的 value 值，所有的交易对都需要

List<AccountType> getAllOtcAccount()获取 全部 的 场外交易账户 
Boolean isOtcAccount(Integer value)根据 value 值判断 是否为场外账户
