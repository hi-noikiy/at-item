交易所 : /order/all_order-获取全部委托

Created by zhangkezhu, last modified on Mar 15, 2018
/order/all_order (post)：已成交和已取消
参数填写类型说明
symbol必填市场标记，btcusdt，详情看下面
pageSize选填页面大小
page选填页码
返回
字段实例解释
code0 
msg"suc"code>0失败
data

如下：

 

side

 下单方向(买/卖)

total_price

 成交总价

created_at

 下单时间

avg_price

 

平均价格

countCoin

 

计价货币

baseCoin

 

base货币

source 订单来源（web/ios..）
type 委托类型（1:限价委托，2:市价委托）

volume

 

下单量

price

 

下单价

status_msg

 

状态描述

deal_volume

 

已成交量

remain_volume

 

未成交量

tradeList

tradeList：

成交订单

volume

 成交量

feeCoin

 手续费币种

price

 成交价

fee

 手续费

deal_price

 成交总价值

type

 类型（买入/卖出）
