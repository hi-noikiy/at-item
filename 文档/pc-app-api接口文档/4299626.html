参数类型说明备注
wsUrl
String
WS URL 
rateObject货币汇率 
marketObject市场列表 
lanObject语言 
defLan
String默认语言 
lanList
Object语言列表 
lanList-
name
String
语言名称名称根据自己语言显示，比如中文”简体中文“，英文：“EngLIsh”
 lanList > id String 语言标识 格式：zh-CN, zh-TW, en-US....
maket_index
String默认市场 
maket>myMarket
Object用户自选市场 
maket>BTC>ETH/BTC
Object货币对Key所有货币对，统一按照对象格式，所有涉及到货不对的key的字段，统一大写
symbolString货币对小写名称 ethusdt
price
Number该货币对价格精度，8：小数点后8位。
name
String该货币对名称，和KEY一样备用
depth
String该货币对三种深度“0.001，0.01，0.1”：按照字符串，中间以英文逗号隔开
volume
Number该货币对量精度8：小数点后8位
rankNumber是否进入首页排行榜 0是 / 1否

仅 h5 首页中用

nc_appkeyString滑动验证key 
default_country_codeString+86默认国家编码
nc_langStringcn滑动验证语言
emailOptCode
Object邮箱编码列表 
smsOptCode
Object短信编码列表 
app_logo_listjson对象 app中需要用到的logo

header里面用户语言key：exchange_language
