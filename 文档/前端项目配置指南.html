链上科技交易所-前端项目配置指南

使用到的技术

  1. NODE.JS (由于涉及大量的 ES6/7 等新属性，node 需要 6.0 以上版本 )
  2. Webpack (项目打包)
  3. Tradingview (K线图)重要：使用Tradingview需要该公司的免费授权，授权流程点此处查看
  4. VUE (技术栈)
  5. echats (深度图)

总结：node + webpack + ES6/7 + vue2 + vuex + vue-router + stylus + echats + svg + TradingView

项目运行

1、安装NODE.JS

2、安装VUE + VUE-CLI脚手架 (参考VUE官网)

3、下载相关源码 (非常重要！！！)

 0、git clone ..../exchange-vue-public ：公共源码，必须！
 1、git clone ..../exchange-vue-ex ：币币交易源码
 2、git clone ..../exchange-vue-co ：合约交易源码
 3、git clone ..../exchange-vue-otc ：场外交易源码

说明：
1、本项目至少需要2个仓库的地址，才能运行起来。
2、其中一个必须是公共源码，1、2、3项目根据具体业务需在0的指定目录下（/src/product/）。
3、例如一个包含了币币交易+场外交易的项目需要：
   在公共源码（编号0的git仓库）的/src/product/这个目录下，Clone2个仓库，分别是：编号1和3的仓库。
   /src/product/这个目录下会新增：/src/product/exchange-vue-ex 和 /src/product/exchange-vue-otc  2个文件夹。

4、npm install 安装相关依赖

5、npm run dev 项目本地运行

npm run dev

6、npm run build 打包发布

npm run build -- ex_text

-ex_text说明：
--：参数
ex:当前打包的是哪个项目，可替换单位：ex、otc、co，不传参数，默认为ex
_text：是否测试域名，如果是测试，而非正式上线，请添加"_text",正式上线，可以不写。
ps：想要自定义域名，请在项目的根目录，打开index.html进行配置。

设置页面Title信息

  • 进入：src/locale/zh.js => pageTitle
  • pageTitle 内有所有的页面的title配置。这是一个语言包，请确保相关的语言包都进行修改。

设置页面皮肤风格

  • 默认皮肤 src/assets/css/variable.styl
  • 白色版皮肤 src/assets/css/variable-white.styl
  • tradingView src/store/trade.js => KlineColor
  • 说明：具体字段说明，请进入该文件下进行参考。
