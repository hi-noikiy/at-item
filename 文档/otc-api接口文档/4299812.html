交易所 : /otc/search - 广告搜索-OK

Created by yulei, last modified by xingmingfeng on Jun 29, 2018
/otc/search（POST）
参数类型必填项说明用例
side String 交易类型（出售 OR 求购）"SELL", "BUY"
symbol String必填交易币种BTC
payCoinString 支付币种CNY
paymentsString 支付方式

"otc.payment.alipay", "支付宝"

"otc.payment.wxpay", "微信支付"

"otc.payment.international.wire.transfer", "国际电汇"

sortString 排序方式

0, "时间降序排序"

1, "价格升序排序"

2, "价格降序排序"

3, "数量升序排序"

4, "数量降序排序"

pageSize number 页大小20
pagenumber 页号1
