交易所 : /otc/person_home_page - 个人主页用户基本信息显示-OK

Created by yulei, last modified by xingmingfeng on Jun 28, 2018
/otc/person_home_page （POST）
参数类型必填项说明用例
uid integer必填被查看的用户id 
返回
字段实例说明
code0请求成功
data

identity： 0 判断此页面访问情况（如下）：

0：未登录用户查看他人的主页和登录用户查看自己的主页；
1：登录用户查看他人的主页，并且当前显示用户在登录用户黑名单中；
2：登录用户查看他人的主页，并且当前显示用户不在登录用户黑名单中

imageUrl：用户头像

otcNickName： 用户昵称

completeOrders：用户订单成交笔数（交易次数）

complainNum：总申诉量

sucComplainNum：胜诉量

loginStatus：在线状态 （1在线，0离线）

registerTime：注册时间

lastLoginTime：最后登录时间

authLevel： 认证状态 0、未审核，1、通过，2、未通过  3、未认证

mobileAuthStatus：是否开启了手机认证:0-未开启,1-开启

请求数据
message '成功'返回消息
