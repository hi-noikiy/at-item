交易所 : /otc/person_ads - 获取个人广告信息-OK

Created by yulei, last modified by xingmingfeng on Jun 27, 2018
/otc/person_ads （GET）
参数类型必填项说明用例
uid integer必填被查看的用户id 
pageSize number必填页大小 
page number必填页号 

新接口返回：

{
    "code": 0,

    "data": {
