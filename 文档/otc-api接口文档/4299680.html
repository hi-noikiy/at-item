交易所 : /otc/wanted_save - 发布广告-OK

Created by yulei, last modified by zhongjingyun on Jul 03, 2018

 

新接口入参：

/otc/wanted_save（POST）
参数类型必填项说明用例
side String必填广告类型'SELL', 'BUY'
coin String必填交易币种'BTC'、'ETH'、'USDT'
payCoinString必填支付币种'CNY'、'USD'
paymentsString必填支付方式

"payments":[

{
"payment":"otc.payment.alipay",
"account":"alipay123"
},
{
"payment":"otc.payment.wxpay",
"account":"wxpay123"
}
],


priceBigDecimal必填单价 
volumeBigDecimal必填交易总数量数量单位默认根据所选币种进行切换
minTradeBigDecimal必填单笔最小交易额(金额) 
maxTradeBigDecimal必填单笔最大交易额(金额) 
dealVolume   Integer必填对方必须交易次数限制 
limitTimeInteger必填交易期限单位：分钟
daysInteger必填广告开放时间单位：日
descriptionString 广告说明（备注） 
autoReplyString 聊天自动回复内容 
priceRate BigDecimal必填溢价比率 固定价格时：0,写整数：80 表示80%
priceRateTypeByte必填交易价格与市场价的比率类型

0：固定价格

2：高于 ,

3：低于.

 

旧接口入参：

/otc/wanted_save（POST）
参数类型必填项说明用例
side String必填广告类型'SELL', 'BUY'
coin String必填交易币种'BTC'、'ETH'、'USDT'
payCoinString必填支付币种'人民币'、'美元'
paymentsString必填支付方式

"payments":[

{
"payment":"otc.payment.alipay",
"account":"alipay123"
},
{
"payment":"otc.payment.wxpay",
"account":"wxpay123"
}
],


priceRate BigDecimal必填溢价比率 
priceRateTypeByte必填交易价格与市场价的比率类型

1：市价 ,

2：高于 ,

3：低于.

priceBigDecimal必填单价 
volumeBigDecimal必填交易总数量数量单位默认根据所选币种进行切换
minTradeBigDecimal必填单笔最小交易额(金额) 
maxTradeBigDecimal必填单笔最大交易额(金额) 
beginDateStringopenType="CUSTOM"时为必填交易开放开始日期 格式：yyyy-mm-dd
endDateStringopenType="CUSTOM"时为必填交易开放结束日期格式：yyyy-mm-dd
limitTimeInteger必填交易期限单位：分钟
termsString 支付说明 
descriptionString 广告说明（备注） 
authTypeString 对家认证级别

'phone': 手机认证,

'email': 邮箱认证,

'c1': 身份认证.

多个用 ',' 隔开

dealVolumeInteger必填对方必须交易次数限制 
beginTimeStringopenType="CUSTOM"时为必填交易开放开始时间格式：hh:mm
endTimeStringopenType="CUSTOM"时为必填交易开放结束时间格式：hh:mm
autoReplyString 聊天自动回复内容 
openTypeString 广告开放方式

'DEFAULT': 默认时间30天随时可见,

'CUSTOM': 用户指定时间段内可见

capitalPwordString 交易密码(已废弃) 
return
字段实例解释
code0成功
data  
msg'string' 
{
  "code": 0,
  "data": {},
  "msg": "string"
}
